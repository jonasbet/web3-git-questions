Review 3 different GUI Git clients

SmartGit

Intuitive UI makes it easy for first time users. Its interface is clean and uncluttered. Sidebars are used to display all repositories.

GitKraken

Excellent for beginner as it is easy to use. It is as intuitive as possible. For example, switching branches just takes a click.

Fork

Implemented by SourceTree, confortably stages changed line-by-line. Of special importance is to compose different line changes is you commits after coding for a while.
